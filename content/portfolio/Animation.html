This project was a small project to test out skeletal animation in OpenGL. The project was made in 2 weeks and was a solo project. The project use a custom engine named WhiteBox it was quite fun to learn Animation in C++ . You can check out the project on GitHub

Key Features

  • Skeletal Animation
  • Mesh Skinning
  • Mathematics library
