ModerOpenGl

Introduction

I started working on the Modern OpenGL Sandbox project to learn more about OpenGL and how to create a game engine from the ground up. We were given two weeks to create a graphics engine that could render Blinn Phung Light and import OBJ files.

You can check out the engine on GitHub

Key Features

  • Modern OpenGL
  • Model Loading
  • Blinn-Phong Lighting
  • 3D Camera
