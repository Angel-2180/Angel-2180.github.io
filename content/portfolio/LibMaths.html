LibMaths

Introduction

LibMaths is a project that I started to learn more about Mathematics and how to make a library as well as helping in my other projects.

The project is a library to see the functionality of Quaternion and Matrix and Vectors as well as somes utils. You can check out the Lib on GitHub

Key Features

  • Quaternion
  • Matrix
  • Vector
  • Utils

Additional Info

This lib is based and tested to follow the GLM library and is used in my other projects. The lib is still in development and will be updated as I learn more about Mathematics and interesting features to add.
