GigEngine

Introducing GigEngine, a three-month school assignment undertaken by groups of three, aimed at crafting a functional engine and leveraging it to create a demo game.

This project embraces a modern approach, incorporating essential features such as a Scene Graph with Quaternions, In-Game UI, Skeletal Animation, Scripting in Lua, Serialization, Physics with Bullet, and an Editor powered by ImGui. GigEngine aims to provide valuable insights into engine architecture and design decisions, making it an invaluable resource for aspiring developers. Explore its features on GitHub .
